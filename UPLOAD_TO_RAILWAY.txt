📋 طريقة التحميل:

Option 1: GitHub (الأفضل - تلقائي)
  1. ارفع الملفات إلى GitHub
  2. Railway سيسحبها تلقائياً
  
Option 2: Railway CLI (محلي فقط)
  1. حمّل المشروع على جهازك المحلي
  2. سجل دخول: railway login
  3. انشر: railway up
  
Option 3: يدوياً عبر الواجهة
  1. افتح Railway Dashboard
  2. اذهب لـ Settings → Variables
  3. عدّل المتغيرات حسب الحاجة
